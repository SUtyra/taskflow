✦ TaskFlow
Минималистичный менеджер задач — учебный проект для освоения стека HTML + CSS + JS + Node.js.

Структура репозитория
taskflow/
├── server/
│   └── index.js          # express-сервер, REST API
├── public/               # cтатика
|   ├── style.css         # стили
│   ├── index.html        # html-страница
│   └── js/
│       ├── api.js        # слой запросов
│       ├── ui.js         # функция рендеринга
│       └── app.js        # логика и обработка событий
├── .gitignore
├── package.json
└── README.md

Что реализовано
_______________________________________________________________________
REST API     | 5 эндпоинтов: GET список, GET по ID, POST, PATCH, DELETE
_______________________________________________________________________
CRUD задач   | Создание, просмотр, редактирование, удаление
_______________________________________________________________________
Статусы      | К выполнению → В работе → Готово
_______________________________________________________________________
Приоритеты   | Низкий / Средний / Высокий
_______________________________________________________________________
Фильтрация   | По статусу через query-параметры
_______________________________________________________________________
Статистика   | Счётчики по каждому статусу
_______________________________________________________________________
Тостер       | Уведомления об успехе/ошибке
_______________________________________________________________________
Адаптивность | Mobile-friendly (≥ 320px)
_______________________________________________________________________
SPA fallback | Express отдаёт index.html для всех маршрутов
_______________________________________________________________________
