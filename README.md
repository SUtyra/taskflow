TaskFlow
Минималистичный менеджер задач. Учебный проект для практики Fullstack разработки на связке HTML + CSS + JS (Frontend) и Node.js + Express (Backend).

Основные возможности
CRUD операции: Создание, чтение, обновление и удаление задач.
Управление статусами: Переключение между состояниями "К выполнению", "В работе", "Готово".
Приоритеты: Назначение уровня важности (Низкий / Средний / Высокий).
Фильтрация: Отображение задач по текущему статусу.
Статистика: Счетчики количества задач для каждого статуса.
Уведомления: Всплывающие тосты (успех/ошибка) для обратной связи.
Адаптивность: Интерфейс корректно отображается на экранах от 320px.
SPA: Одностраничное приложение с удобной маршрутизацией на клиенте.

Технологический стек
Frontend: Чистый JavaScript (ES6+), CSS3, HTML5.
Backend: Node.js, Express.js.
API: RESTful эндпоинты для управления задачами.

API Эндпоинты
Сервер предоставляет 5 базовых REST маршрутов для работы с задачами:

Метод  | Эндпоинт	   | Описание
GET	   | /tasks      | Получить список всех задач
GET	   | /tasks/:id  | Получить задачу по ID
POST   | /tasks	     | Создать новую задачу
PATCH  | /tasks/:id  | Обновить существующую задачу
DELETE | /tasks/:id  | Удалить задачу

Дополнительно:
Фильтрация списка через query-параметры (например, /tasks?status=done).
SPA fallback: все неизвестные маршруты отдают index.html.
